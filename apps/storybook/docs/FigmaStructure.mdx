| ColeГ§ГЈo | ConteГәdo | Modo |

## Sync Figma вҶ” Code

### Code вҶ’ Figma
```bash

```

### Tokens Studio (Git Sync)

Os tokens sГЈo compatГӯveis com o plugin **Tokens Studio for Figma**:

1. Configure o Git Sync apontando para `packages/tokens/src/`
2. O formato DTCG (`$value`, `$type`) Г© reconhecido automaticamente
3. ReferГӘncias como `{color.indigo.500}` sГЈo resolvidas pelo plugin
